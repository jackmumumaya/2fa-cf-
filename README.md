

# Cloudflare Pages 免费一键部署指南

**DEMO演示: 2fa.51xueai.org**

这套全新的 2FA 验证码工具已针对 Cloudflare Pages 平台进行了极致优化。由于它是一个纯前台、无数据库依赖的静态页面，非常适合部署在全球极速且永久免费的 Cloudflare 面板上！


## 获取部署包

你目前看到的这个文件夹（`2fa-cloudflare`）就是完成最终调整后的完整部署包！请先将这整个文件夹打包下载到你的本地电脑桌面上。

文件夹内包含2个核心文件：

1. `index.html` - 经过优化的、专门提取独立域名根路径格式的主程序引擎。
2. `_routes.json` - Cloudflare 专用的伪静态重定向规则。它保证了即便用户访问 `/任意错误乱码密钥`，不仅不会显示 404，还会把他们引回框架并提取该密钥运算出双重验证码！

## 极速免费部署 - 仅需 3 分钟

请按照以下简单步骤操作：

### 第 1 步：登录 Cloudflare 平台

1. 你需要一个免费的 Cloudflare.com 账户（如果有请登录，没有注册一个即可）。
2. 在左侧功能菜单中，找到并点击 **Workers & Pages (工作器与页面)** 选项卡。

### 第 2 步：上传至 Pages (免费页面托管)

1. 在主界面上，点击显眼的蓝色大按钮 **Create application (创建应用程序)**。
2. 切换至顶部右侧的 **Pages (页面)** 选项卡页面。
3. 页面的下半部分会显示一个带有虚线的大虚框区域，写着：**Upload assets (上传资产/直接上传文件)**。
4. **【核心步骤】**：直接把你桌面上我们刚才做好的整一个 `2fa-cloudflare` 文件夹（也就是包含了 `index.html` 和 `_routes.json` 的那个文件夹总体）用鼠标拖拽到这个虚线框里，松手。

### 第 3 步：见证奇迹的时刻

当进度条跑完后（通常不超过几秒钟），Cloudflare 会显示上传并发布成功！
平台会自动为您分配一个类似 `https://51xueai.pages.dev` 属于你自己的高防极速二级域名。

你现在就可以点进去，在网域后面输入你的 2FA 密钥感受秒变验证码顺畅的体验了！

---

## 附加：绑定自定义域名

如果你嫌分配的域名难看，想把它变成类似 `2fa.mcy.com`。

只要你的 `mcy.com` 的 NS 解析在 Cloudflare：

1. 在刚发布成功的项目中，点击头部导航兰切换到 **Custom Domains (自定义域)**。
2. 点击 **Set up a custom domain (设置自定义域)** 按钮。
3. 输入你想要绑定的二级短域名（如果是本号已解析的域名，Cloudflare 还会贴心地自动帮你搞定一切 DNS 记录）。
4. 稍等片刻，它会自动部署好最高级的 SSL 安全证书！

从此，你就拥有了一个属于自己品牌、永不掉线、全球 CDN 加速的极速 2FA 工具了！
